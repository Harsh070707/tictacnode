1. Start the server by writing a command on command line or a terminal - node index.js
2. After that go to the browser pass the url localhost:5050
3. The game will open and wait for the second player to join
4. When second player will join please the game will start 
5. The second player will join the game with same url but from different tab
6. When the both player will join game will start 
7. The first player has first chance to move 
8. And second player will have second chance to move means after first player 
9. If all the 9 cells will filled than the game will tie 
10. Otherwise those who will make possible combination like diagonal ,straight with same symbol X or O 
11. He/She will win the game and the message will display
